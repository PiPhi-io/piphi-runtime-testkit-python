# Changelog

## 0.1.0

Initial public release of the PiPhi Python runtime testkit.

Added:

- pytest plugin registration for PiPhi runtime test fixtures
- mock Core server for capturing telemetry and event requests
- builder helpers for runtime headers, `/config`, and `/config/sync` payloads
- assertion helpers for telemetry, events, and entities responses
- end-to-end FastAPI example coverage for realistic runtime flows
